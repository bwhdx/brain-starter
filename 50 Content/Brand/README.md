# Brand — content operations hub

See `Brand tooling.md` for the indexed entry point.

This folder holds the operational discipline for shipping content:
- Voice formula, pre-publish gate, vocabulary, pillars, channels — the per-post discipline
- Content process, cadence — the workflow and rhythm

The *identity* (who you are, what you stand for) lives in `80 Areas/Identity/`. This folder is the *tooling* for shipping content that sounds like you.
