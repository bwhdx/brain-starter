# Drafts — channel-specific drafts

Each channel has its own subfolder. Drafts live here until they ship.

Status:
- `draft` — body text exists, hasn't been gated
- `ready` — passed the pre-publish gate, ready to copy-paste into the platform
- `killed` — decided not to post

When posted: move file to `50 Content/Published/`.

See `_Meta/Templates/Draft.md`.
