# Templates

Templater templates for every note type. Configure Obsidian → Settings → Templater → Template folder = `_Meta/Templates`. Optionally map folders to default templates so new files in `20 Atoms/`, `40 Sources/Books/`, etc. auto-populate.

## Templates included

| File | Purpose |
|------|---------|
| `Atom.md` | One atomic idea (use sub-kind in frontmatter) |
| `Capture.md` | Quick inbox drop |
| `Daily.md` | Daily note (Daily Notes plugin) |
| `Source-Book.md` | Book in `40 Sources/Books/` |
| `Source-Article.md` | Article in `40 Sources/Articles/` |
| `Source-Talk.md` | Talk in `40 Sources/Talks/` |
| `Source-Podcast.md` | Podcast in `40 Sources/Podcasts/` |
| `Seed.md` | Content seed in `50 Content/Bank/` |
| `Draft.md` | Content draft in `50 Content/Drafts/{channel}/` |
| `Published.md` | Posted content in `50 Content/Published/` |
| `Person.md` | Person in `60 People/{group}/` |
| `Project.md` | Project README in `70 Projects/{name}/` |
| `Decision.md` | Decision in `90 Decisions/` |
| `Goal.md` | Goal in `80 Areas/Goals/` |
| `Area.md` | Area hub doc |
| `Map.md` | MOC in `30 Maps/` |
| `Place.md` | Place in `80 Areas/Travel/` |
| `Weekly review.md` | Sunday ritual checklist |
