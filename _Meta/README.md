# _Meta — vault infrastructure

You don't need to touch this folder day-to-day. It holds the scaffolding.

- `Schema.md` — frontmatter spec for every note type
- `Workflows.md` — technical workflow reference (the daily/weekly/monthly rhythms in operational detail)
- `Plugins.md` — which Obsidian plugins to install and why; the ~10-plugin cap and the rationale
- `Durability.md` — backup strategy (3-tier: Sync + Git + Time Machine)
- `Templates/` — all 18 Templater templates
- `Bases/` — saved Obsidian Bases views (database queries over frontmatter)
