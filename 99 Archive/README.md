# 99 Archive — cold storage

Completed projects, retired content, dormant notes.

## Don't delete — archive

Storage is free. Retrievability is everything. Archive when:
- A project is done or killed (move the project folder here)
- A piece of content is retired (move from `Published/` to `99 Archive/Content/`)
- A note is no longer relevant but you don't want to lose it

## Structure

Loose. Make subfolders as needed:
- `99 Archive/Projects/`
- `99 Archive/Content/`
- `99 Archive/Misc/`

## Don't touch unless asked

The Archive is a different folder for a reason. Don't navigate here for working notes. Search reaches everything anyway.
