# A theme map looks like this

> **EXAMPLE FILE — delete or replace once you have a few real themes.**

A theme map gathers atoms by belief cluster. The graph view is the messy version; the theme map is the curated version.

## When to create one

When you notice 3+ atoms clustering around a theme. Earlier than that is guessing; later than that and the cluster is invisible to retrieval.

## What goes in a theme map

A one-paragraph description of the theme, then a curated list of atoms in it, then "see also" links to related maps or identity docs.
