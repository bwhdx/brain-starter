# 40 Sources — immutable external inputs

External material as you encountered it. Books, articles, talks, podcasts. **Don't confuse highlights with insights.** Insights live in atoms; sources hold the raw material so the lineage is preserved.

## Subfolders

- **Books/** — one note per book. Use the `Source-Book` template.
- **Articles/** — web articles, essays, blog posts.
- **Talks/** — conference talks, lectures, recorded sessions.
- **Podcasts/** — episodes worth keeping.
- **Highlights/** — Kindle / Readwise exports (opt-in; deferred).

## Status lifecycle

- `to-read` — added to the queue
- `reading` — currently consuming
- `read` — finished
- `abandoned` — started and dropped (worth recording why)
- `reference` — never read cover-to-cover but kept for reference

## The extraction loop

1. Add source with `status: to-read`
2. When you start reading: `status: reading`, `date_started: today`
3. As you read: extract *insights* (not highlights) into `20 Atoms/`. Each atom has `source: [[Source name]]`.
4. When done: `status: read`, `date_read: today`, `rating: 1-5`, write a one-paragraph review
