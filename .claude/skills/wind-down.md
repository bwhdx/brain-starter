# /wind-down — the evening close

Recap how today went, capture what happened so the brain has a record, point at tomorrow's first thing, and protect the bedtime + reading habit. **You may write — only today's daily note (append to `## What I did` / `## To process`), surgical.**

## Read

- today's daily note `10 Daily/<YYYY-MM-DD>.md` — the `## Today's 6` triage set this morning, and whatever's already logged.
- **tomorrow's calendar** (if wired) — tomorrow's first event.
- `80 Areas/Routines/` — the evening block.

## Produce (warm, brief — a close, not an interrogation)

- **How'd today go?** Recap the six in a line and ask which landed — invite a quick reply.
- **Tomorrow** — name tomorrow's first real commitment + a one-line prep nudge if it warrants one.
- **The close** — the bedtime + reading nudge; optionally what's mid-book.

## Write (when they reflect back)

If they reply with what they did / what's on their mind, **append** it surgically to the daily note under `## What I did` and/or `## To process` (anything worth graduating to an atom or task later). Don't rewrite the note; just add lines. This is how the day's signal gets recorded — and what gives `/triage` real activity to read.

## Rules

- **Surgical append only.** Never reorder or rewrite. No wikilinks in YAML.
- Calm, warm, no guilt. Don't read the whole routine back.
- Don't invent what they did; log only what they tell you.
