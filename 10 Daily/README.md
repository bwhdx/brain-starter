# 10 Daily — daily notes

One file per day. Format: `YYYY-MM-DD.md`. Raw running log.

## What goes here

- What's on your mind today
- What you did
- What you noticed
- Things to process later

## What this is NOT

- A diary. Don't perform self-reflection. Just capture.
- A polished log. Daily notes are rough. Anything worth keeping graduates to atoms during weekly review.
- A todo list. Use whatever you use for tasks; this is for thinking and capture.

## The daily ritual

- Open today's note (Obsidian: Daily Notes plugin hotkey, usually Cmd-Shift-D)
- Drop bullets throughout the day as things come up
- Don't try to be comprehensive

## File naming

`YYYY-MM-DD.md` — set in Obsidian → Settings → Daily Notes.

## Template

See `_Meta/Templates/Daily.md`. Configure Daily Notes plugin to use it for auto-population.
